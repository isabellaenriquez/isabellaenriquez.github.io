Bookworm Wannabe
Languages: Python, HTML/CSS, SQL
Frameworks: Flask, PostgreSQL
Created for CS50's Web Programming with Python and JavaScript course, Bookworm Wannabe is a book review website with a PostgreSQL-powered database of 5000 books. On Bookworm Wannabe, registered users can view book information, including some data attained via the Goodreads API, write reviews, and read other users' reviews.
